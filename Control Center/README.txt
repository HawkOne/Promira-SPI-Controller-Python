                          Total Phase Control Center
                          --------------------------

Total Phase products:
- Aardvark I2C/SPI Host Adapter
- Cheetah SPI Host Adapter
- Promira Serial Platform

System Requirements:
- Windows 7, Windows 10, or Windows 11
- Ubuntu Linux 18.04 LTS, 20.04 LTS, or 22.04 LTS
- macOS 11 Big Sur, 12 Monterey, 13 Ventura, or 14 Sonoma
- 64-bit operating system on all platforms

On some macOS systems, double-clicking the application icon may not
launch the application.  To launch the application, right-click the
application icon, select "Open" in the pop-up menu, then select "Open"
in the dialog that appears.
